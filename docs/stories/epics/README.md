# Epic Summary Pages

This folder contains summary pages for each epic — a quick-reference view of story status and progress.

## Relationship to Other Docs

| Folder | Purpose |
|--------|---------|
| `docs/prd/epics/` | Full PRD — detailed requirements and scope |
| `docs/stories/epics/` | Summary page — story table and progress tracking (this folder) |

## File Naming

`{EPIC-ID}-{kebab-title}.md`

Example: `TEMPL-001-standardize-go-templates.md`

## What a Summary Page Contains

- Story table with current status (Done / Ready / InProgress / Deferred / Superseded)
- Progress counter (e.g., 4/7 stories completed)
- Next steps
- Links to epic PRD and related architecture decisions
