# Completed Stories

This folder contains stories that have passed QA review — status is **Done**.

Stories are moved here from `docs/stories/active/` after the QA gate returns a PASS or CONCERNS verdict. They serve as a historical record and reference for implemented behaviour.

## File Naming

`{EPIC-ID}.{STORY-NUM}.{kebab-title}.story.md`

Example: `1.1.project-scaffolding.story.md`

## What "Done" Means

A story is Done when:
- All acceptance criteria are verified (checkboxes checked)
- QA gate verdict is PASS or CONCERNS (with notes)
- Code is merged to the topic branch
- Story status field is set to **Done**
