# Epic Definitions (PRD)

This folder contains the full Product Requirements Documents (PRDs) for each epic — the authoritative source of requirements for the epic and its stories.

## Format

Files follow the naming convention: `epic-{EPIC-ID}-{kebab-title}.md`

Each epic PRD contains:
- **Overview** — Business context and goals
- **Scope** — What is in and out of scope
- **Acceptance Criteria** — High-level success conditions
- **Story Breakdown** — List of stories with dependencies
- **Technical Context** — Architectural notes and constraints

## Relationship to Other Docs

| Folder | Purpose |
|--------|---------|
| `docs/prd/epics/` | Full PRD — requirements and scope (this folder) |
| `docs/stories/epics/` | Epic summary pages — progress and story status |
| `docs/stories/active/` | Stories currently being worked on |
