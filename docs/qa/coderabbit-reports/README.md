# CodeRabbit Reports

This folder stores automated code review reports generated by the CodeRabbit CLI before each QA gate.

## Format

Files follow the naming convention: `{storyId}-coderabbit-report.md`

## When Reports Are Generated

1. @dev completes implementation and runs CodeRabbit pre-commit review
2. CodeRabbit output is saved here for traceability
3. @qa references the report during the gate review

## Severity Handling

| Severity | Action |
|----------|--------|
| CRITICAL | Must fix before QA gate (self-healing loop, max 2 iterations) |
| HIGH | Recommend fix, @qa documents in gate |
| MEDIUM | Create tech debt issue, note in gate |
| LOW | Optional improvement, noted in report |
